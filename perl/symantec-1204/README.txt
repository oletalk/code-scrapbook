HOWTO
This should work with Perl5.8 and above.  Simply place the following files in the same directory:
Card.pm
Deck.pm
Hand.pm
HashList.pm
playrummy.pl

And run 'perl playrummy.pl' to begin. 

Options, parameters and further details are in perldoc playrummy.pl.
